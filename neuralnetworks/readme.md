# terms used in neural
CNNs - Convolutional Neural Networks are used for image processing and computer vision tasks
RNNs - Recurrent Neural Networks are used for sequential data processing tasks
LSTMs - Long Short Term Memory are used for sequential data processing tasks
GANs - Generative Adversarial Networks are used for image generation tasks
MLP - Multilayer Perceptron are used for general purpose tasks
Transformers - Transformers are used for sequence to sequence tasks
